# Лабораторная работа №1: Исследование и понимание данных

## Описание
Репозиторий содержит материалы лабораторной работы №1 по курсу "Интеллектуальный анализ данных".
Цель работы: проведение разведочного анализа данных (EDA), выявление проблем качества данных и формирование отчета.

## Структура репозитория

*   `notebooks/01_data_understanding.ipynb` — основной Jupyter Notebook, содержащий код загрузки данных, поиск дубликатов и пропусков, расчет статистик и визуализацию (EDA).
*   `report/quality_report.md` — детальный текстовый отчёт с описанием выявленных проблем (асимметрия, выбросы, расчетные пустоты) и планом предобработки.
*   `data/` — папка с исходным CSV-файлом данных.


## Использование

1. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
2. Запустите ноутбук `notebooks/01_data_understanding.ipynb` для анализа.
